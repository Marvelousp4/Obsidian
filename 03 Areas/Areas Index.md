# Areas Index

Areas are long-term responsibilities. They are not storage buckets. Each area page defines standards, boundaries, active projects, related knowledge, and review prompts.

## Area Dashboards

```dataview
TABLE status AS Status, focus AS Focus, review_rhythm AS "Review Rhythm"
FROM "03 Areas"
WHERE type = "area"
SORT file.name ASC
```

## How To Use Areas

- Use the daily note for capture.
- Use projects for bounded outcomes.
- Use knowledge notes for reusable conclusions.
- Use area pages to decide what standards, projects, or reviews need to change.

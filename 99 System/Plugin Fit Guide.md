# Plugin Fit Guide

These plugins are not equal. Some belong in the core operating system. Some are occasional tools. A few should stay out of the core path unless you have a very specific use case.

## Core Plugins In This Vault

### Homepage

- Opens [[00 Dashboard/Start Here]] on startup
- Use `Homepage: Open homepage` any time you want to reset to the operating hub

### Periodic Notes

- Weekly reviews live in `02 Daily/Weekly`
- Monthly reviews live in `02 Daily/Monthly`
- Use it for review rhythm, not for raw capture

### Kanban

- Use [[00 Dashboard/Weekly Focus Board]] as the weekly execution board
- Best for short-horizon work control, not for long-form knowledge

### Outliner

- Use it in daily notes, meetings, and issue notes
- It makes nested bullets behave like a real outliner, which is better for fast meeting capture

### PDF++

- Use it when reading reports, manuals, specs, and vendor PDFs in `07 Resources`
- Best for linking directly to a page or highlighted region in a PDF-backed source

### Tag Wrangler

- Use only when tags start drifting
- Rename or merge tags from the tag pane instead of manually editing old notes one by one

## Core Plugins Already In Use

- `QuickAdd` for formal capture
- `Templater` for repeatable structure
- `Dataview` for dashboards and indices
- `Tasks` for action tracking
- `Omnisearch` for retrieval
- `Smart Connections` for related-note recall
- `Obsidian Git` for backup and sync
- `Local REST API` for local automation

## Plugins Kept Optional On Purpose

### Copilot

- Do not make this the default AI path unless you intentionally configure a second AI stack
- Your main AI workflow is still daily note plus nightly triage

### Importer

- Use only when migrating from another tool or bulk-importing structured files
- It is not part of normal daily operation

### Mind Map

- Use for occasional concept shaping after a note is already clear
- Do not use it as the primary note-taking surface

### Day Planner

- Useful only if you decide to time-block the day
- It is optional because your current system is input-first, not calendar-first

### Editing Toolbar

- Skip it unless you want a mouse-heavy writing workflow
- On Mac keyboard-driven note-taking, it adds more UI than value

## Default Rule

1. Capture raw material in the daily note
2. Use QuickAdd when a note clearly deserves a formal object now
3. Use Kanban for short-horizon execution
4. Use Weekly and Monthly reviews to compress lessons into direction

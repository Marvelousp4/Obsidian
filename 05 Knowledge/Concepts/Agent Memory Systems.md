# Agent Memory Systems

## Definition

Agent memory systems are mechanisms for long-horizon retention, summarization, retrieval, and selective forgetting in autonomous or semi-autonomous AI workflows.

## Why It Matters

Memory quality controls consistency and long-term performance more than short-context benchmark gains.

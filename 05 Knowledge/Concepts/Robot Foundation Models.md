# Robot Foundation Models

## Definition

Robot foundation models are large pre-trained models used as base priors for perception, planning, and control across multiple robot tasks.

## Why It Matters

They increase baseline capability but create tradeoffs in cost, interpretability, and adaptation speed for site-specific workflows.
